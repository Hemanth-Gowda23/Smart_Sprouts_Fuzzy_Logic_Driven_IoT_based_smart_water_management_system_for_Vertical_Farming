# Smart Sprouts — Fuzzy Logic Driven IoT Smart Water Management System for Vertical Farming

> IEEE Published · KSCST Recognised · Python · IoT · Fuzzy Logic · Raspberry Pi

---

## Overview

Smart Sprouts is an IoT-based system for monitoring and controlling agricultural conditions in vertical farming environments. It uses fuzzy logic to make real-time irrigation decisions based on live sensor data — reducing water waste and improving crop yield without manual intervention.

This project was **co-authored into a peer-reviewed IEEE publication** and **recognised under the KSCST 46th Student Project Programme (SPP)** by the Karnataka State Council for Science and Technology.

---

## Problem It Solves

Traditional irrigation systems either overwater or underwater crops because they use fixed schedules. Smart Sprouts reads real-time environmental data and makes intelligent, dynamic decisions — the way an experienced farmer would, but automated.

---

## How It Works

```
Sensors (soil moisture, temperature, humidity)
        ↓
Raspberry Pi collects real-time data
        ↓
Fuzzy Logic Engine processes conditions
        ↓
Decision: irrigate / hold / adjust
        ↓
Actuators respond (water pump on/off)
```

---

## Features

- Real-time sensor data collection via Raspberry Pi
- Fuzzy logic engine for intelligent irrigation decisions
- Handles uncertainty in sensor readings gracefully
- Reduces water usage compared to fixed-schedule systems
- Designed for vertical farming environments
